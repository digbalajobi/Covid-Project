Pandemic Dashboard

Severe Acute Respiratory Syndrome Coronavirus 2 (SARS-CoV-2) A.K.A. Covid-2019 Novel Coronavirus

Line Graph

Cases By Country

This is an interactive tool that displays the global impact of the Coronavirus pandemic. You are able to physically see the increase in Coronovirus cases within a span of 5 months (January to May of 2020). The drop down on the top left hand of corner contains a list of countries in Alphabetical order which allows for the graph to change by country.

As you can see all the graphs show a positive linear graph representing the increase in Covid over the 5 month period. In order to be able to display this line graph we set the variable country to the first index of the data which represented the all the countries in the dataset. We then used a for loop to grab and iterate through the dates and a corresponding coronavirus value. A d3 select button was added in order to create the drop down that listed all the countriesand a .on(change) function so that the line graphs can change as the different countries are select.
